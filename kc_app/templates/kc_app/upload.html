File Format Requirements
Excel Format Requirements

Microsoft Excel file (.xlsx or .xls). Only the first sheet will be processed.

Required Structure:
  • Headers must be in the first row (row 1)
  • Data starts from row 2
  • Must contain the following columns: id, type, question
  • Options for Multiple Choice Questions should be stored in separate columns, each beginning with "choice_"; Other types of questions should leave the choices columns blank.
Supported Formats:
  • .xlsx (Excel 2007+)
  • .xls (Excel 97-2003)
Example

**Note: Bolded columns are required.

Row 1: |     id     |       type        | question        | choice_a | choice_b | choice_c| answerKey | Misc Fields |
Row 2: | sciqa-2327 |  Multiple Choice  | Which is ...    | Diamond  |   Wool   |   Wood  |     b     |     ...     |
Row 3: | sciqa-4389 | Fill in the Blank | When forces ... |          |          |         |  balanced |     ...     |

CSV Format Requirements

Comma-separated values file with a header row containing column names. Must include a column with your questions.

Required Structure:
  • Header row in the first line
  • UTF-8 encoding recommended
  • Comma as delimiter (standard CSV format)
  • Must contain the following columns: id, type, question
  • Options for Multiple Choice Questions should be stored in separate columns, each beginning with "choice_"; Other types of questions should leave the choices columns blank.
Example:

**Note: Bolded columns are required.

Line 1: id,type,question,choice_a,choice_b,choice_c,answerKey,Misc Fields
Line 2: sciqa-2327,Multiple Choice,Which is ...,Diamond,Wool,Metal,b
Line 3: sciqa-4389,Fill in the Blank,When forces ...,,,,balanced,...

JSON Format Requirements

JavaScript Object Notation file containing an array of objects with consistent structure.

Required Structure:

Fields marked with an asterisk (*) are required.

  • Root element must be an array
  • Valid JSON syntax required
  • Each object must contain the following properties: id, type, question
  • Question text should be stored in a nested object
    • Question text stored as "stem"
    • choices is an array of objects, only included for MCQs
Example:

Note: Bolded fields are required.

[
    {
        "id": "sciqa-2327",
        "type": "Multiple Choice",
        "question": {
            "stem": "Which is more flexible?",
            "choices": [
                {"label": "a", "text": "diamond"},
                {"label": "b", "text": "wool sweater"}
            ]
        },
        "answerKey": "b",
        # Miscellaneous fields
        "skill": "Compare properties of materials",
        "subject": "natural science",
        "topic": "physics",
        "category": "Materials",
            "grade": "grade3"
    },
    {
        "id": "spacing-1161-fib-5",
        "type": "Fill in the Blank",
        "question": {
            "stem": "When forces are _____, an object's motion does not change."
        },
        "answerKey": "balanced",
        # Miscellaneous fields
        "LO": "I can compare balanced vs unbalanced forces",
        "Topic": "6.3 Force and Motion",
    }
]

JSONL Format Requirements

JSON Lines format with one JSON object per line.

Required Structure:

Fields marked with an asterisk (*) are required.

  • One valid JSON object per line
  • No commas between objects
  • Each object should have consistent properties
  • Newline character separates objects
Example:

Note: Bolded fields are required.

# Note for actual jsonl files, each line in the file must be a complete, valid JSON object.
# Each object is expanded here just for readability purposes
{
    "id": "sciqa-2327",
    "type": "Multiple Choice",
    "question": {
        "stem": "Which is more flexible?",
        "choices": [
            {"label": "a", "text": "diamond"},
            {"label": "b", "text": "wool sweater"}
        ]
    },
    "answerKey": "b",
    # Miscellaneous fields
    "skill": "Compare properties of materials",
    "subject": "natural science",
    "topic": "physics",
    "category": "Materials",
        "grade": "grade3"
}
{
    "id": "spacing-1161-fib-5",
    "type": "Fill in the Blank",
    "question": {
        "stem": "When forces are _____, an object's motion does not change."
    },
    "answerKey": "balanced",
    # Miscellaneous fields
    "LO": "I can compare balanced vs unbalanced forces",
    "Topic": "6.3 Force and Motion",
}
