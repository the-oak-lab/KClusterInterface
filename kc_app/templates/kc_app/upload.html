File Format Requirements
CSV Format Requirements

Comma-separated values file with a header row containing column names. Must include a column with your questions.

Required Structure:
  • Header row in the first line
  • At least one column containing questions
  • UTF-8 encoding recommended
  • Comma as delimiter (standard CSV format)
Example:
id,type,question,answerKey
sciqa-2327,Multiple Choice,Which is more flexible? a) diamond b) wool sweater, b

Excel Format Requirements

Microsoft Excel file (.xlsx or .xls). Only the first sheet will be processed.

Required Structure:
  • Headers must be in the first row (row 1)
  • Data starts from row 2
  • At least one column containing questions
  • Empty rows will be ignored
Supported Formats:
  • .xlsx (Excel 2007+)
  • .xls (Excel 97-2003)
Example Structure:
elearning-mcq-1,Multiple Choice Row 1: | id | type | question | answerKey |
Row 2: | elearning-mcq-1 | Multiple Choice | What is ... | b |

JSON Format Requirements

JavaScript Object Notation file containing an array of objects with consistent structure.

Required Structure:
  • Root element must be an array
  • Each object should have consistent properties
  • At least one property containing questions
  • Valid JSON syntax required
Example:
[
  {
    "question": "What is the capital of France?",
    "category": "Geography",
    "difficulty": "Easy"
  },
  {
    "question": "Explain quantum physics",
    "category": "Science",
    "difficulty": "Hard"
  }
]

JSONL Format Requirements

JSON Lines format with one JSON object per line.

Required Structure:
  • One valid JSON object per line
  • No commas between objects
  • Each object should have consistent properties
  • Newline character separates objects
Example:
{
  "id": "sciqa-2327",
  "type": "Multiple Choice",
  "question": {
    "stem": "Which is more flexible?",
    "choices": [
      {"label": "a", "text": "diamond"},
      {"label": "b", "text": "wool sweater"}
    ]
  },
  "answerKey": "b",
  # Miscellaneous fields
  "skill": "Compare properties of materials",
  "subject": "natural science",
  "topic": "physics",
  "category": "Materials",
  "grade": "grade3"
}
